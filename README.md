# Lumina Tinerilor — site static cateheză greco-catolică

Site static (HTML/CSS/JS) pentru un parcurs catehetic de 12 săptămâni adresat tinerilor greco-catolici.

## Pagini
- `index.html` — introducere și call-to-action.
- `resurse.html` — plan pe 12 săptămâni și 6 teme cu idee-cheie, exercițiu, rugăciune.
- `calendar.html` — tabel orientativ cu întâlniri.
- `despre.html` — descriere metodă și scop.
- `contact.html` — formular demo cu confirmare în pagină (fără backend).

## Stil și funcționalitate
- Temă dark, responsive, accent verde.
- Navigație între pagini + footer cu anul curent setat din `script.js`.
- Form de contact cu mesaj de confirmare la submit (nu trimite date).

## Folosire locală
Deschide oricare pagină în browser (dublu-click sau `python -m http.server` dacă preferi un server static). Nu sunt dependențe externe.
